BossFlash is a utility on top of boss_session, you can store messages between requests, automatically populated to the view as undefined and deleted from session after used.

Usage

Add flash messages in the controller like:

boss_flass:add(Req, notice, "Flash Title", "Flash Message")

Use the boss_flash var in the view, should render "notice - Flash Title - Flash Message":

Functions

boss_flash:add/3

add(Req, Type, Title)

boss_flash:add/4

add(Req, Type, Title, Message)
